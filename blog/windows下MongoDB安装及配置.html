windows 下 MongoDB 安装及配置

# 登录 Mongodb 官网下载并安装 MongoDB

https://www.mongodb.com/download-center/community

# 先创建数据库文件的存放位置

在 MongoDB 安装目录下创建 data，在 data 下再创建 db：D:\mongodb-4.0.9\data\db，因为启动 mongodb 服务之前需要必须创建数据库文件的存放文件夹，否则命令不会自动创建，而且不能启动成功。

# 启动 MongoDB 服务

  1. 打开 cmd 命令行
  2. 进入 D:\mongodb-4.0.9\bin 目录（注意：先输入 d:进入 d 盘，然后输入 cd D:\mongodb-4.0.9\bin）
  3. 输入如下的命令启动 mongodb 服务：mongod --dbpath D:\mongodb-4.0.9\data\db
  4. 在浏览器输入http://localhost:27017 （27017 是 mongodb 的端口号）查看，若显示：
    则表示，连接成功。如果不成功，可以查看端口是否被占用。

# 配置本地 windows mongodb 服务

这样可设置为 开机自启动，可直接手动启动关闭，可通过命令行 net start MongoDB 启动，net stop MongoDB 停止。

  1. 在 MongoDB 安装目录下新建配置文件 mongo.config
  2. 用记事本打开 mongo.config ，并输入：
    dbpath=D:\mongodb-4.0.9\data\db
logpath=D:\mongodb-4.0.9\data\log\mongo.log
  3. 用管理员身份打开 cmd:在 C:\Windows\System32 找到 cmd 的运行文件,右键，选择以管理员身份运行.
  4. 配置 windows 服务：cmd 先跳转到 D:\mongodb-4.0.9\bin 目录下。 输入：mongod --config "D:\mongodb-4.0.9\mongo.config" --install --serviceName "MongoDB" 即根据刚创建的 mongo.config 配置文件安装服务，名称为 MongoDB。完成后，查看 windows 本地的服务,可以看到名为 MongoDB 的服务, 表示配置成功.
